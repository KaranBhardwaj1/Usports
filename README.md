🏆 USports – Chitkara University Sports Management System

USports is a smart digital platform developed for Chitkara University to manage sports facilities, equipment, bookings, and live match updates in a centralized system. It connects students, admins, and the sports organization through a real-time and role-based system.

🎯 Objective

To modernize and simplify sports management at Chitkara University by providing a unified platform for equipment management, facility booking, and live sports updates.

🚀 Features
👤 Student / User Panel
View available sports equipment and facility availability in real-time
Book facilities like pool table and other sports areas
View assigned equipment and booking status
Access live match scores and updates
Receive global announcements and notifications
Invite other players for matches via global messaging


🛠️ Admin Panel
Update equipment availability in real-time
Assign equipment to users
Manage and approve booking requests (e.g., pool bookings)
View all active and upcoming bookings
Monitor system activity and user requests
Maintain inventory of sports resources


🏢 Organization / Sports Committee Panel
Create and manage live match scores
Update ongoing match results in real-time
Organize sports events and tournaments
Broadcast announcements to all users
Monitor team participation and performance

💬 Core Functionalities
🔴 Live Score Updates (managed by Organization)
📅 Booking System for sports facilities (Pool table, etc.)
🧾 Equipment Allocation & Availability Tracking (Admin)
🌐 Global Messaging System
👥 Player Invitation System
🔐 Role-Based Access Control (User / Admin / Organization)
🧑‍💻 Tech Stack
Frontend: React / React Native
Backend: Node.js, Express.js
Database: MongoDB
Real-Time Communication: Socket.IO
Authentication: JWT (JSON Web Token)
